# Filters

At any location inside PingBack it is possible to open up the filter view by pressing the `CMD+F / STRG+F` shortcut or the `filter` icon in the top right.

## What you can filter by

### Text
*Included in:*
- eventName
- title
- description

### Projects

<Callout type="default" emoji="ðŸ’¡">
